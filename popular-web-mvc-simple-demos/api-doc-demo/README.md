<center><h1>Popular MVC框架接口文档示例</h1></center>

#### 1、简介

> 此项目用于演示使用popularmvc提供的接口文档自动生成功能，popularmvc已经内置了swagger+knife4j，只要引入了popularmvc依赖，默认就开启了接口文档自动化功能，生产环境可以通过设置`popularmvc.enableSwagger=false`关闭接口文档。主要包含如下内容：
>
> 1. 自动生成的接口文档信息
> 2. 忽略接口文档
> 3. 为接口添加额外的描述信息
>     * 描述controller信息
>     * 描述接口信息
>     * 描述普通参数
>     * 描述域模型参数
